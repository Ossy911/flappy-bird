# Sky Flap - Premium Flappy Bird Clone

A beautiful, single-file Flappy Bird clone built with HTML5 Canvas, CSS, and Vanilla JavaScript.

## Features
- **Responsive Design**: Fully playable on mobile devices, tablets, and desktops.
- **Progressive Difficulty**: The game speeds up and the gaps get tighter as your score increases!
- **Sound Effects**: Simple generated audio feedback for flapping, scoring, and crashing (using the Web Audio API).
- **High Scores**: Automatically tracks your best score in the current session.
- **Modern UI**: Smooth glassmorphism menus with elegant animations.

## How to Play
- **Desktop**: Press the `Spacebar`, `Up Arrow`, or click the mouse to flap.
- **Mobile**: Tap anywhere on the screen to flap.
- Navigate the bird through the green pipes without hitting them or falling off the screen.

## Installation
Since this is a single-file application, no complex installation or build steps are required! 
1. Clone the repository or simply download the `index.html` file.
2. Double-click `index.html` to open it in any modern web browser.

## Technologies Used
- HTML5 Canvas
- CSS3 (Animations, Flexbox, Custom Properties)
- JavaScript (ES6 Classes, RequestAnimationFrame for the game loop)
- Web Audio API (Zero external assets required)
